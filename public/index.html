Projects

ScriptFlow

ScriptFlow is a web-based application that allows users to extract text from images using OCR (Optical Character Recognition) and convert typed text into realistic handwriting using customizable fonts. It supports multilingual input, integrates grammar checking for enhanced accuracy, and includes a drawing board for manual input and export. The tool provides an intuitive interface for seamless image upload, text detection, and handwriting generation.

Blogging Website

Key features include user authentication, rich text editing, and responsive design for a user-friendly blogging experience.

IoT based Air Quality Monitoring System

This project focuses on developing a smart IoT solution to monitor air quality in real-time. Utilizing ESP32, MQ135 (for air quality), and DHT11 (for temperature and humidity) sensors, the system collects environmental data and transmits it to the cloud via Arduino IoT Cloud. An intuitive Android application, created using Arduino's IoT Remote, displays the data using interactive widgets like meters and graphs. This project empowers users with actionable insights into air quality, helping drive awareness and informed decisions for healthier living.
